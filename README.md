# Recipe Management API

This project is a Spring Boot REST API for managing recipes.

## Features
- CRUD operations for recipes
- Categories and ingredients
- Reviews with ratings
- Average rating calculated dynamically
- Validation and exception handling
- DTO & Mapper architecture
- Unit testing for service layer

## Technologies
- Java 17
- Spring Boot
- Spring Data JPA
- H2 Database
- JUnit 5 & Mockito

## Architecture
Controller – Service – Repository with DTO mapping.

## Testing
Unit tests are implemented for the service layer using JUnit and Mockito.

## Note
This project is backend-only and follows an API-first design.
